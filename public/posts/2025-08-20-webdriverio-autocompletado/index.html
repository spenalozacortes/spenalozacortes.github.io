Tener autocompletado en VSCode para WebdriverIO con JavaScript

Dos opciones:

  1. Importar los métodos y objetos globales de WebdriverIO en cada archivo de pruebas:
import { expect, browser, $ } from '@wdio/globals';
  1. Crear un archivo jsconfig.json en la raíz del proyecto y hacer referencia a los paquetes wdio utilizados:
{
    "compilerOptions": {
        "types": [
            "node",
            "@wdio/globals/types",
            "@wdio/mocha-framework"
        ]
    }
}
